This repository contains the code and data for the manuscript "Topological Signatures of Planar Branching Networks: A Comparative Study Across Four Formation Mechanisms." 

## Contents 
- code/pipeline/ -- All Python analysis scripts 
- data/features.csv -- Topological features for 1040 graphs 
- results/figures/ -- Publication-quality figures 
- MANUSCRIPT_COMPLETE.pdf -- Full manuscript
